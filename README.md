# Malaria Prediction and Antimalarial Drug Discovery Projects
This repository contains a collection of machine learning and data science projects focused on malaria research, predictive analytics, and antimalarial drug discovery.It includes work on NDHS-based malaria prevalence modeling, QSAR analysis, and scientific computing using Python.
Projects Included
1. Malaria Prevalence Prediction (NDHS Dataset)
   Machine learning models developed to predict malaria infection patterns using:
   - Nigerian Demographic and Health Survey (NDHS) datasets
   - Features from household surveys, children biomarker data, environmental and demographic predictors
   - Supervised ML algorithms (Random Forest, XGBoost, SVM, Logistic Regression, etc.)
Includes:
- Data cleaning and preprocessing
- EDA and feature engineering
- Model training and evaluation
- Performance interpretation and visualization
